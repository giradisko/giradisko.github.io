## Perché 'giradisko'?

### Perché la 'k'?

- **giradischi** = apparecchio elettrico per la riproduzione sonora di dischi (vinili).

- al singolare perché **sì** 

- la _**k**_ per far riferimento a loro: gli **Skiantos**
